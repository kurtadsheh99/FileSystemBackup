# Security Policy

## Supported Versions

Supported verssions are:

| Version         | Supported          |
| -------         | ------------------ |
| 1.27 (Latest)   | :white_check_mark: |


## Reporting a Vulnerability

to report a vulnerbility, please create an issue with details
on how it was found, what version of OS you were on. 
Screenshots are highly appricated, just make sure you cover 
sensitive information in the screenshot if there is any 😊
